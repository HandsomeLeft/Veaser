# 快速上手

## 尝试一个demo

### 1.安装组件库
```bash
npm i veaser

```
### 2.引用组件库
```javascript
//全部引入
import "veaser/dist/css/index.css";
import Veaser from 'veaser';
Vue.use(Veaser);

//按需引用
import "veaser/dist/css/demo.css";
import {v_demo} from 'veaser';
Vue.use(v_demo)

```
