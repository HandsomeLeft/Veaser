### 介绍

Veaser 一个面向前端初学者的Vue2.x组件库，全面拥抱 `Vue2` 生态，由个人进行维护。

### 特性
- 🚀 提供50+个高质量通用组件 
- 🚀 组件十分轻量
- 💪 由国人开发，完善的中文文档和后勤保障
- 🛠️ 支持按需引入
- 🛠️ 支持主题定制
- 💪 确保90%以上单元测试覆盖率，提供稳定性保证


### 快速上手

### 尝试一个demo

### 1.安装组件库
```bash
npm i veaser

```
### 2.引用组件库
```javascript
//全部引入
import "veaser/dist/css/index.css";
import Veaser from 'veaser';
Vue.use(Veaser);

//按需引用
import "veaser/dist/css/demo.css";
import {v_demo} from 'veaser';
Vue.use(v_demo)

```
